# Python SQLite IN NOT IN Operators

This Python script demonstrates the usage of SQL `IN` and `NOT IN` operators using an in-memory SQLite database. It creates a sample `products` table, inserts data, and then performs various queries to filter results based on category using both operators. It also includes an example of using `IN` with a subquery.

## Language

`python`

## How to Run

1. Save the code as `main.py`.
2. Run from your terminal: `python main.py`
